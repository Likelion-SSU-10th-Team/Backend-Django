## 파일 경로
### `/home`
- 인화 팝업
- blank
- 필름선택 팝업
- 로그인 하기 전 intro
### `/home/print`
- 인화 직전 스크롤 
### `/home/write`
- 일기작성 
### `/read` 
- 일기열람

### `/films` 
- 필름 보관함 
### `/films/{film_id}` 
- 필름 상세보기

### `/albums` 
- 앨범 보관함 
### `/albums/{album_id}` 
- 앨범 상세보기

## 🛠️ 기술 스택
### 백엔드
- 프레임워크: Django
- 데이터베이스: Maria DB
- 배포: 클라우드 타입
- 이미지 저장소: Amazon S3
- 개발 도구: VS code
- 테스트 도구: Postman
